Trie (Prefix Tree)

A Trie, also know as a Prefix Tree, is a tree-like data structure that stores a dynamic set of strings where the keys are usually sequences, such as words. Each node in the Trie represents a character, and the path from the root to a particular node spell out a word. Trie structures are particularly useful for tasks involving dynamic sets of strings and are commonly used in applications like autocomplete systems

In a Trie

  • Each node of the Trie represents a character.
  • The edges between nodes represent the links between characters
  • The path from the root to a specific node forms a string
  • Nodes may have links to other nodes representing the continuation of a string.
